O que são tags HTML

Voltar

São marcações utilizadas para informar ao navegador como será estruturado a página HTML.

Elementos podem ter tags de abertura e fechamento separadas, como a div, ou uma mesma tag ser de abertura e fechamento, como a img.

O que são atributos

Voltar

São utilizados para personalizar as tags, podendo modificar tanto sua estrutura quanto sua funcionalidade.

Existem atributos que tem em quase todos as tags, como:
  • id: um identificador único para uma tag, podendo assim manipular via javascript aquela tag.
  • class: podendo ter mais de uma classe com mesmo nome, é muito utilizado para aplicar estilo via css naquela tag através de um arquivo css ou dentro de uma tag style no html.
  • style: utilizado para aplicar estilos css diretamente na tag, sem a necessidade de um arquivo css ou dentro de uma tag style no html.

Elemento de block (block) x Elemento de linha (inline)

Voltar

Block

Ex: div e suas tags semânticas, p, h1, h2, ...

  • Pode definir uma altura e/ou largura
  • Possui margem superior e inferior
  • Por padrão, sempre ocupará toda a largura do elemento pai
  • Mesmo definindo uma largura, o elemento abaixo nunca subirá para ocupar o espaço vazio

Inline

Ex: span, a , img

  • Não pode definir uma largura e/ou altura
  • Não possui margem superior e inferior
  • Ocupa a largura do contéudo
  • Os elementos inline e/ou inline-block abaixo sobem para ocupar os espaços vazios acima

Inline-Block

  • Podem definir uma largura e/ou altura
  • Os elementos inline e/ou inline-block abaixo sobem para ocupar os espaços vazios acima

Tags Semânticas

São tags que para o html é uma div*, mas servem para guiar o desenvolvedor, criando uma semântica no código.

*div é um elemento de bloco que serve para encapsular outros elementos.
  • header: encapsular o cabeçalho
  • nav: encapsular o menu de navegação
  • article: encapsular o artigo em uma página
  • main: encapsular o conteúdo principal da página
  • aside: encapsular um canteúdo lateralmente
  • footer: encapsular o rodapé

Outras tags

  • p: adiciona um parágrafo
  • h1, h2, h3, h4, h5, h6: adiciona um título na página
  • ol: lista ordenada
  • ul: lista não-ordenada
  • li: item de uma lista
  • blockquote: citação.
  • img: adiciona uma imagem através do atributo src
  • input: criar um caixa de entrada de dados
  • a: adiciona um link através do atributo href
  • br: quebrar uma linha
  • i: aplicar itálico no texto
  • strong: aplicar negrito no texto
  • b: aplicar negrito no texto
  • sup: colcoar acima o texto
  • sub: colocar abaixo o texto
  • hr: cria uma linha demarcadora
  • mark: destacar o texto com uma cor
  • pre: criar textos pre-formatados, no qual é bastante utilizado para inserir códigos 
     function javaScript() { ... }
  • del: informa que aquele trecho foi deletado, riscando o texto
  • abbr: sublinhar uma palavra abreviada, como HTML

Atributos

Voltar

  • href: inserir um link na tag a
  • src: inserir em qual local, podendo ser um link ou na própria pasta do projeto, esta a imagem
  • title: titulo ao passar o mouse emcima
  • alt: definir um texto ao falhar a imagem
  • height: definir uma altura para o elemento
  • width: definir uma largura para o elemento
